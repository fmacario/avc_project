Introduction

StrokeRehab é um plataforma para apoio a reabilitação cognitiva maioritariamente a doentes com acidentes vasculares cerebrais (AVC) entre outros.

Nesta página estão descritos os passos necessários para testar os templates teste funcionais desenvolvidos até ao momento. Para aceder ás áreas descritas deverá navegar através da barra lateral de navegação.

Existem actualmente 3 secções principais:

  • Gestão de Pacientes: Nesta área é possivél inserir/remover pacientes na base de dados, associar/desassociar templates a pacientes e verificar dados estatisticos individuais.
  • Área de Configuração: Esta área está destinada á configuração de templates para posterior teste na Área de Exercicio.
  • Área de Exercicio: Esta área existe só para zona de teste de templates previamente configurados na Área de Configuração.
